# analise-carteira-clientes
# Análise de Carteira de Clientes

Este projeto apresenta um dashboard de inteligência de dados focado na saúde financeira e perfil de crédito de uma base de 9.684 clientes.

# Tecnologias e Processos

- Processamento de Dados: Utilizei Python para limpeza e tratamento, e Excel para organização e estruturação final da base.
- Visualização: O dashboard foi construído no Power BI, focando em clareza visual e cruzamento de indicadores.
- Distribuição: Exportação de relatórios em formato PDF para facilitar o compartilhamento executivo.

# Insights Extraídos

- Baixa Inadimplência: Identificou-se que 9.619 clientes não possuem pendências financeiras.
- Perfil de Hipoteca: Dos clientes analisados, 5.260 possuem hipoteca, sendo que apenas 37 deste grupo apresentam endividamento.
- Relação de Empréstimos: Existe um grupo estratégico de 3.794 clientes que mantêm hipotecas ativas sem a necessidade de realizar novos empréstimos.
